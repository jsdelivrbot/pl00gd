COMMANDS

 

 

 

Manager Bouncer User

 

    
!afklimit
 
sets the maximum afk time 
!clearchat
 
clears the chat
 
!cycle
 
toggle DJ cycle
 
!cycletimer
 
set the maximum DJ cycle time for when cycleguard is enabled
 
!locktimer
 
set the maximum time the waitlist can be locked if lockguard is enabled
 
!refresh
 
refreshes the browser of whoever runs the bot (kills the bot as well!)
 
!usercmdcd
 
set the cooldown on commands by grey users
 
!usercommands 
toggle user commands
 
!add @user
 
add user to the waitlist
 
!afkremoval
 
toggles the afk check
 
!alertbot
 
alerts the bot manually
 
!autoskip
 
skips songs automatically when they're done
 
!available
 
marks the bot as available
 
!away
 
marks the bot as AFK and sets the status to away
 
!deletechat @user 
delete all the chats by a certain user
 
!lock
 
lock the waitlist
 
!lockdown
 
lock down the room: only staff can chat
 
!maxlength X 
specify the maximum length a song can be when timeguard is enabled
 
!move @user X
 
moves user to position X on the waitlist, default is position 1 
!remove @user 
remove user from the waitlist 
!roulette
 
start a roulette
 
!songstats
 
toggle song statistics
 
!unlock
 
unlock the waitlist
 
!welcome
 
toggle the welcome message on user join
 
!active X
 
shows how many users chatted in the past X minutes. If no X specified, 60 is set as default
 
!ban @user
 
bans user for 1 day
 
!english
 
ask user to speak english
 
!jointime @user
 
shows how long the user has been in the room
 
!kill
 
shut down the bot
 
!lockskip (reason) 
skip the song and move the dj back up. Lockskip's include: theme, op, history, mix, sound, offensive, nsfw, unavailable, stupid, other
 
!unban @user
 
unban user
 
!lockskippos x
 
set the position to which lockskip moves the dj
 
!cookie @user
 
give a cookie to user
 
!dc @user
 do dc lookup for user 
!eta
shows when user will reach the booth 
!join
 
join the roulette if it's up 
!leave 
leave the roulette if you joined 
!link 
give a link to the current song
 
!love (@user) 
 tells user how much you love them
 
!op
 
links to the OverPlayed list
 
!shoe (@user) 
throws shoe at user
